## :wrench: Prerequisites and Setup

To install all dependencies simply run:

```bash
npm i
```

### Running / Development

You can get app up and running with simple command:

```bash
npm start
```

## :question: Testing

Run:

```bash
npm test
```

## :satellite: Deployment on Gh-Pages

First, run the build:

```bash
npm run predeploy
```

Then run deployment:

```bash
npm run deploy
```
